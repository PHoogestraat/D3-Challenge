National Healthcare Metircs:

Age, Income, Health Risks, and Poverty.

The data set included with the exercise is based on 2014 ACS 1-year estimates from the [US Census Bureau]. The current data set includes data on age, income, smoking, obesity, poverty, healthcare, and poverty by state from 2014.

The graph investigates relationships between Poverty and Healthcare by state. Each bubble represents a state with addtional infromation after clicking on the bubble.
